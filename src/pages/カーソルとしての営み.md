# カーソルとしての営み

カーソルとしての営み
[![カーソルとしての営み](https://i.gyazo.com/efb4e8ea81327f0a44830ad65febc98f.png)](https://gyazo.com/efb4e8ea81327f0a44830ad65febc98f)

それぞれの要素は並列で実行され、依存しあっている。

- Input
  - 人間の入力を範囲で表現する
  - 検索、サジェスト、リンクで範囲を変化させられる
    - 情報処理
- Focus
  - 目標を定めるためにはアフォーダンスが鍵
  - 他者のカーソルを同期的に表示する
  - HTTPによって複数人の行動を同期することができる
    - Websocket, Polling
    - ソーシャル・ネットワーク・サービス
- Action
  - どのように目標に到達するか
  - 他者のカーソルの動きに注目する
  - 快適な道具を用いる
    - 目標によって適切な道具が異なる
    - アプリケーション
      - テキストエディタ
      - ビジュアルツール
